Tentang Slasher.com

Slasher.com lahir dari komunitas gamer yang ingin pengalaman beli game itu simpel, rapi, dan aman. Fokus kami hanya satu: membantu kamu menemukan game terbaik tanpa drama.

Final Project ini mengimplementasikan konsep e-commerce modern dengan modul katalog, keranjang, checkout, dan dashboard admin menggunakan PHP native dan MySQL.
