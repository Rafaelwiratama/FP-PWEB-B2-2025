# Final Proyek Pemrograman Web B2 - Website E-Commerce Game

### Slashers.com

#### Pendahuluan

Slasher.com merupakan aplikasi web berbasis PHP yang dikembangkan sebagai Final Project mata kuliah Pemrograman Web. Website ini berfungsi sebagai platform e-commerce penjualan game digital, lengkap dengan sistem manajemen produk, keranjang belanja, checkout, integrasi payment gateway Midtrans, serta manajemen user dan admin.

Proyek ini dibangun dengan konsep full-stack web development, mencakup frontend, backend, database relasional, serta integrasi API eksternal.
